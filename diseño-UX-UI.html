¿Que és el Diseño UX-UI?

El término UX viene de User Experience, o Experiencia del Usuario. Es cómo el usuario se siente mientras usa cualquier producto o servicio digital que una empresa ofrece. UX tiene que ver también con la sensación que experimenta el usuario después de usar tu producto
